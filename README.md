# req-tracker

Property maintenance request tracker built with React, Node.js, and MySQL.

## What it does

Tenants submit maintenance requests through a form. Property managers can view all open requests, assign contractors, and update the status of each job. Everything is filterable by property and urgency level.

## Stack

- React (frontend)
- Node.js + Express (backend)
- MySQL (database)
- Tailwind CSS

## Features

- Tenant request submission form
- Manager dashboard with filtering
- Contractor assignment per request
- Status tracking: open, assigned, in progress, resolved# req-tracker
